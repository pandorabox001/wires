The General Discussion Forums & Celebrity Forums are 2 separate Forums


--> Gen Forums - can be created by anyone who is logged in as a normal user


--> Celeb Forum - can only be created by the Admin User / the one who is granted access


--> Creating threads in the Forums & Replying to the discussions can be done on any forum by any normal user who is logged in


--> There is an Upvote & Downvote for a particular thread which can be performed by logged in User.


--> Forums can be created for public / group of people (friends / custom )


--> One can view the Forum posted for public view even before the login but to participate in discussion must sign up/login


--> The Forums can be sorted based on different options such as - All Forums, Popular, Oldest, Newest, More views, Trending, etc.


--> The Breadcrumps changes of each of the page as the directed flow from one page to other.
